Random sample consensus (RANSAC)

The RANSAC (Random Sample Consensus) algorithm is a robust and widely used method for fitting models to data contaminated with outliers. Its key strength lies in its ability to handle datasets that contain a significant amount of noise or outliers. The algorithm operates by iteratively selecting random subsets of data points, fitting a model to each subset, and then evaluating the quality of the model by counting the number of inliers, which are points that fit the model well.

How was it done?

To start, the necessary packages and libraries required for the algorithm's implementation are imported. These packages and libraries provide the tools and functions needed to work effectively with the point cloud data.

Next, the RANSAC function is defined, which is the core of the algorithm. The point cloud data is taken as a numpy array, along with the initial number of iterations (N) and the threshold criteria for inlier point selection. The inlier points are determined based on their distance from a fitted plane, as specified by the threshold criteria.

Within the RANSAC function, three points are randomly selected from the entire point cloud to create an initial plane. The orthogonal distance between this plane and each point in the point cloud is then calculated. Points that fall below the specified threshold are considered inliers and their indexes are added to a list.

After processing all the points in the point cloud, a list of indexes representing the inlier points is obtained. These inlier points correspond to the best fitting plane within the point cloud.

To improve the algorithm, an adaptive sampling approach is implemented. This approach calculates the number of iterations (N) based on the outlier ratio and the desired probability (p) of obtaining at least one successful sample. The sample count starts at zero and increments by one in each iteration. The algorithm continues to run in a while loop until the value of N becomes smaller than the sample count, indicating the likely discovery of the best fitting plane.

In this case, a sample size of three points (s) is used, and the probability of at least one successful sample (p) is set to 0.99. These parameter values ensure a high likelihood of accurately identifying the best fitting plane within the point cloud.

By employing this custom RANSAC algorithm, efficient analysis of point cloud data and identification of planes can be achieved. This algorithm finds applications in various fields such as computer vision, robotics, and 3D reconstruction.
